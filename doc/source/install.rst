Installation
============

Basic Installation
------------------

.. code-block:: bash

    git clone https://github.com/FirefoxMetzger/ropy.git
    cd ropy
    pip install -e .


Building the Docs locally
-------------------------

.. code-block:: bash
    git clone https://github.com/FirefoxMetzger/ropy.git
    cd ropy
    sphinx-build -b html doc/source doc/build
